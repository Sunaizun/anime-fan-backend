# Anime Fan Backend API

Backend REST API for an Anime Fan Website. Features:
- User registration & login with JWT
- Protected profile endpoints
- CRUD for Reviews and Posts
- External Anime data via Jikan API
- MongoDB Atlas + Render deployment

## Tech Stack
- Node.js, Express.js
- MongoDB + Mongoose
- JWT (jsonwebtoken)
- bcryptjs
- Render (deployment)
- Jikan API (external API)
